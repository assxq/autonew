영상의 소리 없는 구간을 자동으로 잘라내 새 영상을 만드는 맥용 스크립트를 받습니다. 프리미어 무음 제거 단계를 대체해요. 브라우저에서는 영상을 직접 자를 수 없어, 맥 터미널에서 실행하는 방식입니다.

📖 사용법

  1. 위 버튼으로 remove_silence.js를 받아 영상과 같은 폴더에 둡니다.
  2. 터미널을 열고 그 폴더로 이동합니다. (cd 후 폴더를 터미널에 끌어다 놓기)
  3. node remove_silence.js 영상파일.mp4 를 실행합니다.
  4. 같은 폴더에 영상파일_무음제거.mp4가 만들어집니다.
처음 한 번 ffmpeg 설치가 필요해요: 터미널에 brew install ffmpeg
⚠ 작업 순서가 중요해요
① 영상 렌더 → ② 무음 제거(여기) → ③ 무음 제거된 영상으로 캡컷/브루 자막(STT) → ④ 자막 보정
무음 제거를 자막 뽑기보다 먼저 해야 싱크가 맞습니다. 자막을 먼저 뽑으면 무음 제거 후 다 어긋나요. (영상·음성은 함께 잘리니 둘이 어긋날 걱정은 없어요)
API 키를 넣으면 자동 생성과 내보내기를 쓸 수 있어요.
